## Data Dictionary

Component 5 in the Digital Adaptation Kit (DAK) for Surveillance outlines the minimum set of data corresponding to different points of the workflow within the identified business processes. The data set can be used on any software system and lists the data elements relevant for service delivery and executing decision-support logic, and for populating indicators and performance metrics.
